The following lines are explanation of a project I have been working on and still being developed by adding new infrastructure and features.

THE PROJECT

As the title says this projects is about deploying an AWS EKS cluster, cluster addons, VPC and IAM roles using IaC with Terraform.

AWS EKS

  1. Creates an EKS cluster with specified Kubernetes version.
  2. Configures authentication and access control using IAM roles.
  3. Adds EKS-managed add-ons like CoreDNS, VPC CNI, and AWS EBS CSI Driver
  4. Manages node groups (on-demand and spot instances) for cost optimization.
